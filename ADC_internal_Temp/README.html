MSPM0G3507 Temperature Measurement Using IQMath Library

Overview

This project demonstrates how to measure temperature using the MSPM0G3507 microcontroller, an ADC module, and the IQMath library for precise mathematical operations. The temperature is displayed in both Celsius and Fahrenheit, printed to the console using printf().

Components

  • MSPM0G3507 Launchpad
  • Temperature sensor (internal/external)
  • Code Composer Studio or any compatible IDE

Function Explanation

Function: temperature()

The temperature() function performs the following operations:

  • Initialization: Sets up the ADC and enables interrupts for temperature measurement.
  • ADC Conversion: Starts the ADC conversion process and waits for the result to be available.
  • Voltage Calculation: Converts the ADC result to a corresponding voltage value.
  • Temperature Calculation: Utilizes calibration data to calculate the temperature in degrees Celsius and Fahrenheit.
  • Console Output: Prints the calculated temperature values to the console for monitoring.
  • Delay: Implements a delay to manage the frequency of temperature readings.
