Temperature Display with I2C LCD Using MSPM0G3507

Project Overview

This project demonstrates how to use the MSPM0G3507 microcontroller to read temperature data using the onboard ADC and display it on a 16x2 I2C LCD. The temperature is shown in both Celsius and Fahrenheit.

Features

  • Reads temperature data from a sensor using ADC.
  • Displays the temperature on a 16x2 I2C LCD.
  • Real-time temperature updates in both Celsius and Fahrenheit.

Hardware Requirements

  • MSPM0G3507 LaunchPad (or compatible)
  • 16x2 I2C LCD (Address: 0x27 or 0x3F)
  • Temperature Sensor (internal or external)
  • Connecting wires

Wiring Diagram

Use the following connections between MSPM0G3507 and the I2C LCD:

MSPM0G3507 Pins I2C LCD Pins
SDA SDA
SCL SCL
VCC VCC
GND GND

Code Overview

The project consists of the following key functions:

LCD_init(); // Initializes the I2C LCD in 4-bit mode
temperature(); // Reads temperature from ADC and displays on the LCD
I2C_writeByte(); // Sends data to the I2C LCD

How to Run

  1. Connect the I2C LCD and temperature sensor as shown in the wiring diagram.
  2. Use Code Composer Studio to compile and upload the code to the MSPM0G3507 LaunchPad.
  3. The LCD will start displaying the temperature in both Celsius and Fahrenheit.
